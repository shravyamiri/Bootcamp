# Markdown Cheatsheet

A quick reference guide for writing Markdown syntax.

## Headings

Use `#` for headings. The number of `#` symbols determines the heading level.

```markdown
# Heading Level 1
## Heading Level 2
### Heading Level 3
```

## Lists

### Unordered List

```markdown
- Item 1
- Item 2
  - Subitem 2.1
  - Subitem 2.2
```

### Ordered List

```markdown
1. First item
2. Second item
   1. Subitem 2.1
   2. Subitem 2.2
```

## Links

```markdown
[OpenAI](https://www.openai.com)
```

## Images

```markdown
![Sample Image](img.png)
```

## Code Blocks

Use triple backticks (\`\`\`) to create code blocks. Specify the language for syntax highlighting.

```js
// JavaScript Example
function greet(name) {
  console.log("Hello, " + name + "!");
}
```

## Tables

```markdown
| Name     | Age | City       |
|----------|-----|------------|
| Alice    | 25  | New York   |
| Bob      | 30  | San Diego  |
| Charlie  | 28  | Chicago    |
```
